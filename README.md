## フォーマット

verify 用ファイルをフォーマットしたときにヘッダファイルの順番が異なる現象が確認されています．
これは Clang-Format の仕様であり，詳しくは[こちら](https://google.github.io/styleguide/cppguide.html#Names_and_Order_of_Includes)を参照してください．
